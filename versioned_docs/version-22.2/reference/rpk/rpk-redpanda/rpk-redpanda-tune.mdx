Sets the OS parameters to tune system performance.

Available tuners:

- all
- net
- cpu
- clocksource
- transparent_hugepages
- disk_scheduler
- disk_nomerges
- disk_write_cache
- fstrim
- ballast_file
- disk_irq
- aio_events
- swappiness
- coredump

To learn more about a tuner, run `rpk redpanda tune help |tuner name|`.

### Usage

```
rpk redpanda tune <list of elements to tune> [command] [flags]
```
