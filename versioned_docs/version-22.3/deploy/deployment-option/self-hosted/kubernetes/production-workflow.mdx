The production deployment tasks involve Kubernetes administrators (admins) as well as Kubernetes users.
The admin prepares the cluster with the prerequisites and tunes the worker nodes.
The user deploys and configures Redpanda.

1. Admin: [Prepare the Kubernetes cluster](../kubernetes-cluster-requirements).

1. Admin and User: [Review the best practices](../kubernetes-best-practices).

1. Admin: [Tune the worker nodes](../kubernetes-tune-workers).

1. User: [Deploy Redpanda](../kubernetes-deploy).
