- [Authentication](../console/authentication)

   Authenticate through OAuth 2.0 and OIDC for external identity providers or use plain login.

- [Authorization](../console/authorization)

   Use role-based access control (RBAC) to restrict system access to authorized users.

- [GitHub SSO Setup](../console/github)
  
    Integrate with GitHub to allow users to use their GitHub identities to sign-in to Redpanda Console.

- [Google SSO Setup](../console/google)

    Integrate with Google to allow users to use their Google identities to sign in to Redpanda Console.

- [Okta SSO Setup](../console/okta)

    Integrate with Okta to allow users to use their Okta identities to sign in to Redpanda Console.

- [Generic OIDC](../console/generic-oidc)

    Configure the generic OIDC provider to integrate with an OpenID Connect-compatible identity provider.

- [Plain](../console/plain)

    Sign in to Redpanda Console using a pre-defined set of username/password combinations.
