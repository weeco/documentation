Start, stop and query Redpanda's self-test. Self-test runs benchmarks to characterize disk and network performance of Redpanda machines. It runs through the Admin API listener.

### Usage

```
rpk cluster self-test [command]
```

Available commands:

```
start       Starts a new self test run
status      Queries status of current self test run or returns last results
stop        Stops the currently executing self test
```
