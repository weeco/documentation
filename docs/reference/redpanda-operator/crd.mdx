After you install the Redpanda Operator, you apply the CRD. The CRD has two main components: the basic configuration for the Redpanda cluster, which includes the cluster name and namespace, and the configuration for the APIs.

See the [Redpanda Operator Custom Resource Definition](https://doc.crds.dev/github.com/vectorizedio/redpanda).
